# QUBO++

C++ library and Python package (PyQBPP) for constructing and solving QUBO/HUBO combinatorial optimization problems.

- **Documentation**: https://qubo-plus.github.io/
- **Releases**: See [Releases](https://github.com/qubo-plus/qbpp/releases) for binary packages (.deb, .tar.gz)
- **Install**: `sudo apt install qbpp` (see [Installation Guide](https://qubo-plus.github.io/INSTALL.html))
- **PyQBPP**: `pip install pyqbpp`
